# Dataset

Dataset sử dụng trong project này là **CIC-IDS2017** từ Kaggle.

## Tải dataset

1. Vào link: https://www.kaggle.com/datasets/chethuhn/network-intrusion-dataset
2. Bấm Download → Giải nén
3. Đặt tất cả 8 file CSV vào thư mục `data/`
